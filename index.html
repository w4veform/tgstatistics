⚙️ Настройки синхронизации

Чтобы снапшоты видели все — подключи JSONBin.io:
1. Зайди на jsonbin.io → создай аккаунт (бесплатно)
2. Создай новый Bin (кнопка NEW BIN) → скопируй Bin ID
3. В профиле скопируй Master Key
4. Вставь оба значения сюда. Только у тебя будет Master Key — ты можешь добавлять снапшоты. Остальные только смотрят.
